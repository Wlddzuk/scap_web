Installation

  1. Drag the button above to your bookmarks bar
  2. No bookmarks bar? Press Cmd+Shift+B (Mac) or Ctrl+Shift+B (Windows)
  3. Navigate to any article page you want to capture
  4. Click the "Capture Signal" bookmark
  5. The control room opens automatically with your content ready for processing
Requirement: The SCAP server must be running at localhost:5050 before using the relay.
